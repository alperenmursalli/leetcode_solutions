## 🧠 Patterns (Based on Sean Prashad)

I follow a pattern-based approach using:
https://seanprashad.com/leetcode-patterns/

Instead of solving problems randomly, I focus on mastering patterns.

### Key Patterns

- HashMap / Hashing → O(1) lookup problems
- Two Pointers → sorted arrays, linked lists
- Sliding Window → subarray / substring problems
- Binary Search → sorted input / search space
- DFS / BFS → trees, graphs, grids
- Stack → next greater element, parentheses
- Heap → top K problems
- Dynamic Programming → optimization problems
- Backtracking → permutations, subsets
